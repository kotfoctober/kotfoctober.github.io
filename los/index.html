Load on Start

A lightweight Git quick-reference CLI for learners

Download & install — pick one, or just grab los (works on Intel & Apple Silicon):

  1. 1. Download

    curl -LO https://ericadperkins.com/public/los/los-macos-arm64.bin
curl -LO https://ericadperkins.com/public/los/los-macos-x86_64.bin
curl -LO https://ericadperkins.com/public/los/los.bin

  2. 2. Make executable

    chmod +x los-macos-arm64.bin
chmod +x los-macos-x86_64.bin
chmod +x los.bin

  3. 3. Move into your PATH

    mv los-macos-arm64.bin /usr/local/bin/los
mv los-macos-x86_64.bin /usr/local/bin/los
mv los.bin /usr/local/bin/los

If you’re unsure which file to choose, download los.bin. It works on both Intel and Apple Silicon Macs.
